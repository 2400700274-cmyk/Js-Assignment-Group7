##Project description
This project is a simple interactive e-commerce website built using html,css and vanilla javascript. It allows
users to browse products, search, and filter products, add items to cart, update quantities, remove items, and complete checkout with form validation.
##Technologies used
-HTML
-CSS
-JAVASCRIPT
## How to run the project
1. Open the repository
2. Download the project
3. Open'index.html' in a browser
4. Use navigation links to test cart and checkout pages.
